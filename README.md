
# My Portfolio Website

A personal portfolio I built as part of my MERN stack course. It shows my projects, skills, and experience, and has a working contact form that saves messages to MongoDB.

## Tech Used

- React.js (frontend)
- Node.js + Express (backend)
- MongoDB (database)
- Three.js for the 3D animations
- Framer Motion for scroll animations
- Deployed on Vercel (frontend) and Render (backend)

## Running Locally

Clone the repo, then:

```bash
# backend
cd backend
npm install
npm run dev

# frontend (new terminal)
cd frontend
npm install
npm run dev
```

You'll need a `.env` file in the backend folder with your MongoDB URI:
